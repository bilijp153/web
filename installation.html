安装指南

按照步骤操作，将macOS完美安装到您的ASUS FL8700JP笔记本电脑

准备工作

在开始安装之前，请确保您已准备好以下物品和条件：

所需物品：
  • 16GB或更大容量的U盘
  • 可正常运行的Windows或macOS系统
  • 稳定的网络连接
  • 至少50GB的可用磁盘空间
  • ASUS FL8700JP笔记本电脑 (X509JP)
重要提示： 安装过程中会格式化硬盘，请务必备份所有重要数据。本指南仅适用于ASUS FL8700JP (X509JP)特定机型。
支持版本： macOS Catalina (10.15.x), macOS Big Sur (11.x), macOS Monterey (12.x), macOS Ventura (13.x), macOS Sonoma (14.x), macOS Sequoia (15.x), macOS Tahoe (26.x)

BIOS设置

在开始安装之前，请确保您的BIOS已更新到最新版本，并按照以下步骤进行设置：

1
进入BIOS设置

开机时反复按F2键进入BIOS设置界面。

2
安全启动设置

在Security选项卡中，将Secure Boot设置为Disabled。

3
启动选项

在Boot选项卡中，将Fast Boot设置为Disabled，将CSM设置为Enabled。

4
DVMT预分配

在Advanced选项中，将DVMT Pre-Allocated设置为64M。

5
其他设置

禁用CFG Lock、VT-d和 Thunderbolt安全选项（如果存在）。

6
保存设置

按F10保存设置并退出BIOS。

注意： 不同版本的BIOS界面可能略有不同，但关键设置选项名称基本相同。如果找不到某个选项，请查阅您的BIOS手册或更新BIOS版本。

创建安装介质

1
下载macOS镜像

从苹果官网或可信来源下载macOS安装镜像。推荐使用gibMacOS工具下载。

在Windows上，可以使用gibMacOS工具：

# 下载gibMacOS
git clone https://github.com/corpnewt/gibMacOS
cd gibMacOS
python gibMacOS.command
2
准备U盘

使用磁盘工具将U盘格式化为Mac OS扩展（日志式）格式，GUID分区图。

3
创建安装盘

使用终端命令或BalenaEtcher将镜像写入U盘。

在macOS终端中使用以下命令：

sudo /Applications/Install\ macOS\ [版本名称].app/Contents/Resources/createinstallmedia --volume /Volumes/[U盘名称] --nointeraction

在Windows上，可以使用BalenaEtcher或TransMac工具。

提示： 创建安装介质的过程可能需要30分钟或更长时间，请确保电源稳定且不要中断过程。

安装过程

1
启动安装

从U盘启动，进入OpenCore引导界面，选择安装macOS。

在启动时按ESC键选择启动设备，选择UEFI:您的U盘名称。

2
磁盘工具

进入安装程序后，使用磁盘工具将目标硬盘格式化为APFS格式。

选择"显示所有设备"，选择主硬盘，点击"抹掉"，格式选择APFS，方案选择GUID分区图。

3
开始安装

选择刚格式化的磁盘作为安装目标，开始安装过程。

安装过程会重启数次，每次重启后请继续选择从U盘启动，然后选择macOS安装器继续安装。

4
完成安装

系统会自动重启数次，最终进入macOS设置向导。

按照向导完成macOS初始设置，创建用户账户等。

常见问题： 如果安装过程中出现卡住、重启或错误，请尝试使用不同的引导参数，如"-v"调试模式、"agdpmod=pikera"解决显卡问题等。
