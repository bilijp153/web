安装指南

一步步指导您在 ASUS FL8700JP 上安装 macOS

首页 > 安装指南

重要警告

安装黑苹果存在风险，可能导致数据丢失或硬件损坏。操作前请务必备份重要数据，并确保您了解每一步操作的含义。本指南仅适用于 ASUS VivoBook FL8700JP (X509JP) 特定机型。

准备工作

所需工具和文件

  • 16GB 或以上容量的 U 盘
  • 另一台可用的电脑（Windows/macOS）
  • macOS 系统镜像（推荐最新稳定版）
  • 本项目的 EFI 文件
  • BalenaEtcher 或类似镜像写入工具

注意事项

确保您的 FL8700JP 型号为 X509JP，其他型号可能不兼容。建议在开始前阅读完整的安装指南。

BIOS 设置

在开始安装前，需要进入 BIOS 进行以下设置：

1

进入 BIOS

开机时连续按 F2 键进入 BIOS 设置界面

2

安全设置

在 Security 选项卡中：

  • 禁用 Secure Boot
  • 设置 Administrator Password（可选但推荐）
3

启动设置

在 Boot 选项卡中：

  • 禁用 Fast Boot
  • 设置 Launch CSM 为 Disabled
4

高级设置

在 Advanced 选项卡中：

  • 设置 DVMT Pre-Allocated 为 64M
  • 设置 CFG Lock 为 Disabled（如可用）

完成设置后，按 F10 保存并退出 BIOS。

安装步骤

步骤 1：下载必要文件

访问项目发布页面下载最新版本的 EFI 文件：

下载 EFI 文件

同时下载所需的 macOS 系统镜像。

步骤 2：创建安装介质

1

准备 U 盘

将 U 盘插入电脑，确保其中没有重要数据（将会被完全擦除）

2

写入镜像

使用 BalenaEtcher 或其他工具将 macOS 镜像写入 U 盘：

  1. 打开 BalenaEtcher
  2. 选择下载的 macOS 镜像文件
  3. 选择您的 U 盘作为目标
  4. 点击 Flash 开始写入
3

替换 EFI

写入完成后，使用磁盘工具挂载 U 盘的 EFI 分区，删除其中的 EFI 文件夹，替换为下载的本项目 EFI 文件。

步骤 3：提取本机 ACPI 表

使用 SSDTTime 工具提取您电脑的特定 ACPI 表：

# 在 Windows 或 Linux 环境下运行
python SSDTTime.py -dataset

将生成的 SSDT 文件放入 EFI/OC/ACPI 目录中，替换原有的文件。

步骤 4：开始安装

1

启动安装

插入 U 盘，开机时按 ESC 键选择从 U 盘启动。在 OpenCore 引导界面选择"Install macOS"。

2

磁盘工具

进入 macOS 恢复环境后，使用磁盘工具格式化目标硬盘：

  • 选择 APFS 格式
  • 使用 GUID 分区图
  • 命名卷标为"Macintosh HD"（或其他名称）
3

安装系统

退出磁盘工具，选择"安装 macOS"，按照提示完成安装过程。系统会自动重启多次。

步骤 5：后期配置

1

迁移 EFI

安装完成后，使用 MountEFI 或其他工具将 U 盘中的 EFI 文件夹复制到系统硬盘的 EFI 分区。

2

SMBIOS 配置

在 config.plist 中设置合适的 SMBIOS 信息（推荐 MacBookPro16,2），并使用 GenSMBIOS 生成三码。

3

驱动更新

进入系统后，安装必要的 Kext 驱动，确保所有硬件正常工作。

故障排除

常见问题解决方案

1

无法启动安装程序

检查 BIOS 设置是否正确，特别是 DVMT 设置。尝试使用不同的 USB 端口。

2

安装过程中卡住

尝试添加引导参数 -v 查看详细日志，或在启动时选择安全模式。

3

硬件不工作

检查是否使用了正确的驱动，或尝试更新到最新版本的 EFI。

更多问题请参考 常见问题页面 或在项目 GitHub 页面提交 Issue。
