关键硬件详情

处理器 (Intel i7-1065G7)

支持状态：完美

Ice Lake 架构的 10nm 处理器，4核8线程，基础频率 1.3GHz，最大睿频 3.9GHz。在黑苹果系统中：

  • CPU 电源管理完全正常，支持各种节能状态
  • 睿频功能工作正常，性能发挥充分
  • 温度监控准确，风扇控制正常

核显 (Intel Iris Plus G7)

支持状态：基本支持

集成在 Ice Lake 处理器中的 Gen11 核显，性能较前代有显著提升。目前状态：

  • 驱动正常，支持硬件加速和 Metal API
  • 显示输出正常，支持 HiDPI 分辨率
  • 限制：由于平台限制，HDMI 输出无法工作
  • 解决方案：可通过 USB-C 转 HDMI 适配器实现外接显示

无线网络 (Intel Wireless-AC 9461)

支持状态：部分支持

Intel 无线网卡，支持 802.11ac 标准。通过 itlwm 驱动实现基本功能：

  • Wi-Fi 连接稳定，支持 2.4GHz 和 5GHz 频段
  • 支持 WPA/WPA2 加密
  • 限制：不支持苹果的 Continuity 功能（隔空投送、接力等）
  • 替代方案：可更换为博通网卡以获得完整功能

驱动信息

本项目使用以下驱动确保硬件兼容性：

  • Lilu - 核心插件加载器
  • WhateverGreen - 显卡驱动修复
  • AppleALC - 声卡驱动
  • VirtualSMC - 模拟 SMC 芯片
  • itlwm - Intel 无线网卡驱动
  • VoodooI2C - 触控板驱动

所有驱动均已针对 FL8700JP 硬件进行优化配置，确保最佳兼容性。
