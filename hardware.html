硬件组件 状态 说明
CPU (Intel Core i7-10510U) ✅ 完美支持 所有核心与线程正常工作，支持电源管理
GPU (Intel UHD Graphics) ✅ 完美支持 支持 4K 输出，硬件加速正常
内存 (DDR4) ✅ 完美支持 最大支持 16GB，双通道正常识别
Wi-Fi / 蓝牙 (Intel AX201) ✅ 完美支持 需使用 AirportItlwm 驱动，蓝牙需配合 BlueToolFixup
音频 (Realtek ALC256) ✅ 完美支持 耳机、扬声器、麦克风均正常工作
触控板 (ELAN I2C) ✅ 完美支持 支持多点触控、手势操作
键盘背光 ⚠️ 部分支持 可通过快捷键调节亮度，但无法自动感应环境光
HDMI 输出 ✅ 完美支持 支持 4K@60Hz 外接显示器
USB 接口 ✅ 完美支持 所有 USB 2.0/3.0 接口均可正常识别
摄像头 ❌ 不支持 目前尚无可用驱动，建议使用外接摄像头
指纹识别 ❌ 不支持 macOS 不支持 Windows 笔记本指纹模块

重要提示

以上兼容性基于 OpenCore 1.0.5 及 macOS Sonoma/Tahoe 测试结果。不同 BIOS 版本或硬件批次可能存在细微差异，请以实际测试为准。

如遇硬件异常，请先确认是否使用了本项目提供的最新 EFI 配置，并检查 BIOS 设置是否符合安装指南要求。
