Hackintosh FL8700JP

高级配置

🔋 电池状态显示

已通过 SMCBatteryManager 正确注入，电池百分比和充电状态正常显示。

🌡️ CPU 温度与风扇

使用 VirtualSMC + ACPI 补丁实现温度读取。风扇由 BIOS 控制，macOS 无法调节转速（正常现象）。

💤 睡眠与唤醒

支持睡眠（S3），但部分用户报告唤醒后屏幕闪烁。建议使用“显示器睡眠”代替整机睡眠。

🔒 安全启动

需在 BIOS 中：
- 关闭 Secure Boot
- 启用 AHCI（非 RAID）
- 禁用 Fast Boot
- 设置 OS Type 为 "Other OS"

🔄 更新建议

每次 macOS 大版本更新前，请：
- 备份 EFI 分区
- 暂勿升级 OpenCore，除非有新补丁
- 更新后若无法启动，可临时从 U 盘引导修复
